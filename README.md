# Taskhub - Forsaken
## Features
- Killer & Survivor and Supports ESP.
- Auto Generator (bugged but work).
- Fullbright & NoFog.
- Anti - Admin.
- User-Friendly UI.
## Executor Functions
- getsenv() - highly recommended
- hookfunction() or oth library - highly recommended
- require() - highly recommended
- compareinstances() - mid recommended
- setclipboard() - slightly recommended
- sethiddenproperty & gethiddenproperty - mid recommended
- cloneref() - slightly recommended
> If your executor does not support getsenv or hookfunc, then the script will have limited functionality.
